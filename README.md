# Portfolio (Vite + React)

## Local development

- Install: `npm install`
- Dev server: `npm run dev`
- Production build: `npm run build`
- Preview build locally: `npm run preview`

## Deploying to Vercel

This project is configured for a static SPA deploy on Vercel.

- Framework preset: Vite
- Build command: `npm run build`
- Output directory: `dist`
- SPA routing: all non-file routes rewrite to `index.html` (see `vercel.json`)
